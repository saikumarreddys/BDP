1. Make Directory in Hadoop File System using "-mkdir":
          hdfs dfs -mkdir /files/

2. upload both files to HDFS using the command " -copyFromLocal":
         hdfs dfs -copyFromLocal /home/cloudera/ICP1/shakespeare.txt  /files/
         hdfs dfs -copyFromLocal /home/cloudera/ICP1/word_list.txt /files/

3.Append both the files using appendFromLocal command:
	hdfs dfs -appendToFile /files/shakespeare.txt /files/word_list.txt
